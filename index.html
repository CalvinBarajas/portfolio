AlphaBack Mobile App

This edutainment mobile app was developed in order to help English speakers learn the alphabet backwards in a fun and game-like environment. Learning this skill can help anyone (a) search through a dictionary or long list faster, (b) pass a field sobriety test, and (c) impress friends and family with the ability to recite the alphanet backwards as fast as you can forwards.

Programmable Robot Arm With Vision

This was one of the first Raspberry Pi 3B programmable robot arms with vision on YouTube. Calvin's project was particularly unique in that it used the 74LS241 octal tri-state buffer with high-end Dynamixel servos and OpenCV library.

Northern California Whole House Fans
